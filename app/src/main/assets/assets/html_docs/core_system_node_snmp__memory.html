NAME

snmp__memory - Munin plugin to monitor memory usage of a remote host via SNMP.

CONFIGURATION

The following configuration variables are used

 host      - SNMP host to contact (default taken from link name)
 port      - SNMP port to use (default 161)
 community - SNMP community string to use (default "public")

NOTES

Based on snmp__df plugin.... If this plugin reports different numbers from the snmp_winmem plugin it must be due to snmp impementation quirks....
